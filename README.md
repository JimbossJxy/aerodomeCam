# aerodomeCam

## What is aerodomeCam

AerodomeCam is an opensource camera set up which one can set up at an airport/airfield which will automatically record the surrounding aerodome. It will display a view of the surrounding aerodome as well as when aircraft come in to land/take off it will auto track them. The camera can be set up to then livestream on any public live streaming service or private.

## What is the purpose of this?

This project was created to make it more accessible for pilots to get an understanding of what traffic is like at the airport, weather conditions & what aircraft are in use. A current solution from what I believe does not exist which runs in some what real time. It will also combine a multiple services like FR24 and LiveATC into a more compact model. It will not directly use FR24 and LiveATC however.

## What features are included?

- Multiview of the surround airport
- Live tracking of aircraft
- Real time radio calls on either single or multiple frequencies
- Display of surrounding aircraft using ADSB
- Making it modular for your needs, i.e. you only need multiview of the airport, you can choose the multicam only. Or if you need the tracking plus adsb only then you can have that instead
- And so much more to come

## Why make it opensource

I don't want any money from it as there is no point. I enjoy aviation and this is something I want to make to better help others and also make it a bit more interesting at ***your*** local airport. Also if others have cool features or something interesting to add it makes it easier, also then there are no real big maintance costs.
